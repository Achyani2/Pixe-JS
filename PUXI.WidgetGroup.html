Class: WidgetGroup

WidgetGroup

A widget group is a layout owner that can position its children according to the layout given to it.

new PUXI.WidgetGroup ()

puxi.mjs:1711
Example
```
const group = new PUXI.InteractiveGroup();

group.useLayout(new PUXI.AnchorLayout());

group.addChild(new PUXI.Button({ text: "Hey" })
 .setLayoutOptions(
     new PUXI.AnchorLayoutOptions({
            anchorLeft: 100,
            anchorTop: 300,
            anchorRight: .4,
            anchorBottom: 500,
            horizontalAlign: PUXI.ALIGN.CENTER
     })
 )
)
```
